# Wordpress Snippet

1. Avatar Single: Permite substituir o avatar padrão (Gravatar) do usuário com ID especifico por uma imagem personalizada hospedada localmente no site, sem o uso de plugins.
2. Avatar Multplie: Permite substituir os avatares padrão (Gravatar) dos usuários com IDs específicos por imagens personalizadas hospedadas localmente no site, sem o uso de plugins.
3. Smart WEBP Converter: Converte automaticamente imagens GIF, JPEG e PNG para WebP usando a biblioteca GD, ajustando a qualidade entre 95% e 80% para reduzir o tamanho sem perda perceptível. Substitui o original apenas se o WebP for menor.
